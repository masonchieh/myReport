📋 專案摘要

專案名稱： TP-Link HS300 整合

目的： 切換電源開關，控制板卡

技術：

  • Web APIs
  • 可直接與設備溝通，溝通方式使用 XOR 加解密協議，包裝 JSON 數據，與官方 APIs 互動
  • 目前通訊方式混用，部分透過 Edward Wu，部分透過官方 APIs
  • 實作一個 rust module 與之通訊互動

重點必要行為：

  • 設置孔位上下電
  • 取得孔位當下電流
  • 取得/監聽設備基本資訊

注意事項：

  • 同一 Port 的 開on / 關off 行為, 時間間隔不可太短, 最好等待至少 3 秒
  • 不同 Port 不會有 on / off 的時間間隔限制

完成狀態：

  • 讀取電流資訊，輔助判斷是否已經順利完成切換板卡
  • 問板卡狀態是否 ready 才是重點，已經 ready 才能開始進入執行測試狀態
  • 前端 vue 顯示開關狀態
  • 後端 rust 控制板卡
  • 測試

重要提醒：

⚠️ 目前暫停開發
