  • 目的：自動化同步 前後端程式 之間的資料傳遞介面，避免人為輸入錯誤
  • 流程：
    1. 先定義 backend rust struct
    2. 透過 schemars、serde_json crate，將 rust struct 轉換成 .json
    3. 再將 .json 轉換成 .ts
      1. 執行 npx json2ts -i schema/Greet.json -o src/schema/Greet.ts
      2. json2ts 套件執行檔需先透過 npm global 安裝, npm install -g json-schema-to-typescript
    4. 最後將產生的 .ts 複製/貼上 至 frontend vue3 使用
  • 如果介面有異動，最好都從 backend rust align 至 frontend vue3，避免前後端介面不一致而無法傳遞資料
